React: 
React é uma biblioteca JavaScript para construir interfaces de usuário. Desenvolvido pelo Facebook, ele permite a criação de componentes reutilizáveis para construir interfaces de usuários interativos.

Diferença entre biblioteca e Framework: 
Uma biblioteca é um conjunto de funções e rotinas que podem ser reutilizadas por um programa, enquanto um framework é uma estrutura mais abrangente que define a estrutura de um programa e dita como certas tarefas devem ser realizadas.
Link: https://www.dio.me/articles/framework-vs-biblioteca-qual-a-diferenca-e-como-escolher-a-melhor-opcao-para-seu-projeto

Props: 
Props, ou propriedades, são utilizados para passar dados de um componente pai para um componente filho no React. São usados para transmitir informações entre componentes.
Link: https://www.freecodecamp.org/portuguese/news/react-js-para-iniciantes-props-e-state-explicados/#:~:text=React%20para%20iniciantes.-,O%20que%20s%C3%A3o%20props%3F,pai%20para%20o%20componente%20filho).

Hooks: 
Hooks são funções especiais no React que permitem o uso de estado e outras características do React em componentes funcionais, que anteriormente eram limitados aos componentes de classe.

Componentes: 
Componentes são blocos de construção reutilizáveis no React, que podem ser compostos para formar interfaces de usuário. Eles podem ser de dois tipos: funcionais (baseados em funções) ou de classe.
Link: https://pt-br.legacy.reactjs.org/docs/components-and-props.html#:~:text=Conceitualmente%2C%20componentes%20s%C3%A3o%20como%20fun%C3%A7%C3%B5es,que%20deve%20aparecer%20na%20tela.
