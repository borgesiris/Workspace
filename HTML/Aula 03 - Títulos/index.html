HTML
Hyper Text Markup Language
É o esqueleto de uma página , uma linguagem ou conteúdo especial que os computadores entendem,
projetado para descrever páginas de web e tornar o conteúdo interativo.
CSS
Cascade Styling Sheets
É a estilização de uma página, define a aparencia e o estilo da página,
desde cores, fontes, posicionamento dos elementos e animação de tais.
JavaScript
É o pai de tudo, nele se consegue fazer alterações no HTML e CSS.
